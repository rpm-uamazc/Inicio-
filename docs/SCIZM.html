Sistema de Consulta de Información de Zonas Metropolitanas (SCIZM)

¿Qué es el SCIZM?

Es una opción que te permite conocer aspectos socidemográficos y económicos sobre cada una de las 76 zonas metropolitanas de México.

¿Qué espero encontrar aquí?

Puedes elegir de una lista de claves alguna de las 76 zonas metropolitanas. Cada opción abre una nueva página con información especifica de la zona elegida. Además, encontraras el link para poder bajar y consultar la base completa por tu cuenta.

Selecciona una zona metropolitana

Si no conoces las claves de la zona metropolitana que buscas, !da clic aquí!.

Aqui puedes consultar toda la información disponible en el sitio sobre zonas metropolitanas. |
